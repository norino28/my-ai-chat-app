# AI クイズ・なぞなぞチャットボット

## プロジェクト概要

家族みんなで楽しめるAIクイズ・なぞなぞチャットボット。
AIがクイズやなぞなぞを出題し、ユーザーがチャット形式で回答する。

## 技術スタック

| 役割 | 技術 |
|------|------|
| フロントエンド | Next.js 15（App Router） |
| バックエンドAPI | Hono |
| ORM | Prisma（MongoDB adapter） |
| AIフレームワーク | Mastra |
| AIモデル | Claude（Anthropic） |
| データベース | MongoDB |

## 機能要件

### 必須機能
- AIがクイズ・なぞなぞをランダムに出題する
- ユーザーがチャット形式で回答を入力する
- AIが正誤を判定し、解説を返す
- 次の問題に進む

### 対象ユーザー
- 家族全員（子供から大人まで）
- ログイン不要（認証なし）
- 会話履歴の保存なし（セッション内のみ）

## 画面仕様

### チャット画面（メイン画面）
- 吹き出し形式のチャットUI（LINEやChatGPT風）
- AIの発言：左側の吹き出し
- ユーザーの発言：右側の吹き出し
- テキスト入力欄と送信ボタン

## アーキテクチャ

```
[ ブラウザ ]
    ↓ HTTP
[ Next.js App Router（フロントエンド） ]
    ↓ API呼び出し
[ Hono（APIサーバー / Next.js Route Handler） ]
    ↓
[ Mastra Agent ]
    ↓
[ Claude API（Anthropic） ]
```

## Mastraエージェント仕様

- エージェント名：`quizAgent`
- 役割：クイズ・なぞなぞの出題、正誤判定、解説
- システムプロンプト方針：
  - 家族全員が楽しめる表現（やさしい日本語）
  - 問題は段階的に難しくしない（ランダム出題）
  - 正解・不正解どちらでも次の問題に誘導する

## 開発ルール

### コードスタイル
- インデント：スペース2つ
- コメント：日本語
- 変数名・関数名：英語
- 関数は単一責任の原則に従い、小さく保つ

### テスト
- 意味のあるアサーションのみ記述する
- 境界値・異常系もテストする
- テストケース名は何をテストしているか明確に記述する

### セキュリティ
- APIキー等の機密情報は`.env`で管理、コードに直接書かない
- `.env`は`.gitignore`に追加する

### 作業スタイル
- 作業はステップごとに進め、各ステップ完了時に報告する
- 大きな変更前に内容を説明してから実行する
- 不明点があれば作業前に確認する

### エラー対応
- エラーが発生した場合は原因と解決策を日本語で説明する
- 同じエラーを3回試みても解決しない場合は作業を止めてユーザーに報告する
